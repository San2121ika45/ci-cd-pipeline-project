# CI/CD Pipeline using GitHub Actions

## Project Overview
This project demonstrates a real CI/CD pipeline using GitHub Actions. The pipeline automatically deploys a Professional Portfolio Website to GitHub Pages whenever code is pushed to the main branch.

## Technologies Used
- Git
- GitHub
- GitHub Actions
- YAML
- GitHub Pages

## CI/CD Workflow
1. Developer writes code in VS Code.
2. Code is committed and pushed to GitHub.
3. GitHub Actions automatically starts.
4. The workflow validates and deploys the website.
5. GitHub Pages publishes the latest version.
